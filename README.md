# Marketing-Campaigns
Identify significant variables impacting customer acquisition and evaluate marketing campaign effectiveness using data-driven insights
This project analyzes a marketing mix dataset to understand key factors influencing customer acquisition. Performed exploratory data analysis (EDA) and statistical hypothesis testing across the four Ps of marketing: People (demographics), Product (spending behavior), Place (sales channels), and Promotion (campaign performance).
The analysis includes data cleaning, feature engineering, and visualization to uncover patterns in customer behavior and evaluate the effectiveness of different marketing strategies.
